Memory should be able to show where it came from.

AIppocampus helps an agent recover prior source instead of pretending the model itself remembered. Summaries, scents, and semantic maps can help with navigation, but source remains the ground.

Source first Clean source and explicit references outrank generated summaries.
Quiet by default Memory should surface when it helps, not turn every prompt into a personal profile.
Local first Conversation source, registry rows, indexes, and sync bundles are private by default.

Continuity is a human problem before it is an engineering one.

The work began with the feeling that a bright new session can still be strangely newborn. AIppocampus gives a future agent a way to return to the thread honestly, with the path, the source, and the limits still visible.

Agent first-use path

Probe without registering private history, then ask for consent before source import or hook setup. 

uvx aippocampus onboard --provider codex --status
# after consent:
uvx aippocampus onboard --provider codex --all
uvx aippocampus search "a distinctive old phrase"

Not innate memory. Not a cloud profile. A way back.

AIppocampus is early, careful infrastructure for continuity across threads, projects, devices, and ordinary life context. It is broader than coding memory, though coding agents are a clean pressure test. The public project keeps claim boundaries explicit: source-backed recall is not the same thing as a model magically remembering you.
